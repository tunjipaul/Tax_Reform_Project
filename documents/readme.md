The **capstone project specification** notebook that describes the entire Agentic RAG challenge. Contains:

- **The Challenge** — Why the Tax Reform Bills 2024 Q&A Assistant matters
- **Your Mission** — What you'll build (AI engine, backend, frontend)
- **Problem Statement** — Nigerian Tax Reform Bills context (confusion, misinformation, real impact)
- **Technical Stack** — LangChain, LangGraph, FastAPI, React, Chroma vector DB
- **What You'll Build** — Three-component system (AI engine, backend, frontend)
- **Group Requirements** — Team formation, deliverables, grading rubric
- **Evaluation Criteria** — 100-point rubric covering AI quality, backend, frontend, docs, code quality
- **Submission Details** — Timeline, deadline (Wed, 7 Jan 2026), presentation date (Fri, 9 Jan 2026)

**Read this first to understand the project scope and requirements.**
